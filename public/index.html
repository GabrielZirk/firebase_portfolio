Run run run // 01.2021

Thought it would be fun to link two things I quite enjoy: doing sports and data analysis. So I created a dashboard to analyse my runs. For data collection I used the myWorkouts App and a Wahoo belt to record my heart rate. Data analysis/manipulation was done with pandas, plotting with plotly/dash. The project is not yet finished, I'm going to continously improve the plots, especially the layout.
